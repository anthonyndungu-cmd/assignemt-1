Intranet

What it is

An intranet is a private network that uses internet technologies but is accessible only to authorised users, usually within an organisation. It looks and works like the internet — web pages, links, file sharing — but it is closed off from the public.

What it is used for

Intranets are commonly used for internal communication, document sharing, employee directories, and other organisational purposes.Companies use intranets to share internal documents, post announcements, manage HR records, host internal tools, and enable communication between employees. Think of it as a private website for a company's staff.

Intranet vs Internet

An intranet is a private network accessible only to authorised users within an organisation, while the internet is a public network accessible to anyone with an internet connection.The internet is public and global. An intranet is private and restricted. While the internet connects billions of people worldwide, an intranet connects only the people within a specific organisation. Access is typically controlled through login credentials or VPN.
