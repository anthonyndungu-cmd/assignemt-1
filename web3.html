Web 3.0

What it is

Web 3.0 is used to describe two distinct visions that are sometimes conflated. The original meaning, proposed by Tim Berners-Lee, refers to the Semantic Web — a web where data is structured in a way that machines can understand it, enabling smarter search and data interoperability. The more recent usage refers to a decentralised web built on blockchain technology.

The Semantic Web Vision

Berners-Lee imagined web pages tagged with metadata that machines could read and interpret. Instead of just displaying text for humans, pages would expose structured data that computers could process. Technologies like RDF, OWL, and SPARQL were developed toward this goal.

The Blockchain/Decentralisation Vision

The more recent Web 3.0 concept focuses on removing central authorities from the web. Instead of your data living on Facebook's or Google's servers, it would live on decentralised networks — blockchains, peer-to-peer systems. Cryptocurrencies, NFTs, and decentralised apps (dApps) are associated with this vision.

Where Things Stand

Web 3.0 in both senses remains partially realised. The Semantic Web never fully took off at scale. The blockchain version is active but faces real questions about usability, energy consumption, and regulatory uncertainty.
