HTTP (Hypertext Transfer Protocol)

What it is

HTTP is the protocol that defines how browsers and web servers communicate. When you visit a website, your browser sends an HTTP request to the server. The server processes it and sends back an HTTP response — usually an HTML file

HTTP Methods

HTTP uses different methods depending on what the client wants to do. GET retrieves data. POST submits data to the server. PUT updates existing data. DELETE removes data. These are the core methods behind most web applications and APIs.

HTTP vs HTTPS

HTTPS is the secure version of HTTP. It uses TLS (Transport Layer Security) to encrypt data between the browser and server. This prevents third parties from intercepting sensitive information like passwords or credit card numbers. You can tell a connection is secure by the padlock icon in the browser's address bar.

Status Codes

HTTP status codes are three-digit numbers that indicate the outcome of an HTTP request. Some common status codes include:

  • 200 OK: The request was successful.
  • 404 Not Found: The requested resource could not be found.
  • 500 Internal Server Error: An error occurred on the server.
