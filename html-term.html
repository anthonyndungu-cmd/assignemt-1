HTML (HyperText Markup Language)

What It Is

HTML is the standard language used to create web pages. It uses tags to structure content — telling the browser what is a heading, what is a paragraph, what is a link, and what is an image. HTML is not a programming language; it is a markup language. It describes structure and meaning, not logic.

Basic Structure

Every HTML document has a standard structure: a DOCTYPE declaration at the top, an html element wrapping everything, a head section for metadata (title, links to stylesheets), and a body section for visible content. Tags come in pairs — an opening tag and a closing tag — though some are self-closing.

HTML and CSS and JavaScript

HTML handles structure. CSS handles appearance. JavaScript handles behaviour. These three technologies are the foundation of every website. An HTML file can link to external CSS and JavaScript files, or include them inline.

HTML Versions

HTML has evolved over the years. HTML5, released in 2014, is the current standard. It introduced new semantic elements (like header, nav, article, footer), native audio and video support, and APIs for things like geolocation and local storage.
