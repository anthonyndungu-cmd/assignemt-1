Web Servers

What it is

A web server is a computer (or software running on a computer) that stores website files and responds to requests from browsers. When you access a website, your browser is talking to a web server

How Web Servers Work

The server listens continuously for incoming HTTP requests. When a request arrives, it locates the relevant file — an HTML page, image, video, or other resource — and sends it back to the browser. For dynamic websites, the server may also run code (PHP, Python, Node.js, etc.) to generate a response before sending it.

Examples

  • Apache HTTP Server
  • NGINX
  • Microsoft IIS
  • Google App Engine
