h1>TCP/IP (Transmission Control Protocol/Internet Protocol)

What it is

TCP/IP is the set of rules that governs how data moves across the internet. It is actually two separate protocols that work together. IP handles addressing and routing — making sure data knows where to go. TCP handles delivery — making sure data arrives completely and in the right order.

How it works

When you send data over the internet, it gets broken into small chunks called packets. Each packet travels independently across the network, possibly taking different routes. When they all arrive at the destination, TCP reassembles them in the correct sequence. If any packets are missing or corrupted, TCP requests them again.
