DNS (Domain Name System)

What it is

The Domain Name System translates human-readable domain names (like google.com) into IP addresses (like 142.250.190.46) that computers use to identify each other. Without DNS, you would have to type a numerical IP address every time you wanted to visit a website.

How Resolution Works

When you type a domain into your browser, your computer first checks its local cache. If the address is not stored there, it contacts a DNS resolver — usually provided by your ISP. The resolver queries a chain of DNS servers: root servers, then top-level domain servers (for .com, .org, etc.), then authoritative name servers for the specific domain. The final answer — the IP address — is returned to your browser
